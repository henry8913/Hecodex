# Hecodex - Shopify Theme

A modern, responsive Shopify theme designed for gaming and tech e-commerce stores.

## Theme Information

- **Theme Name:** Hecodex
- **Version:** 1.0
- **Author:** Henry G.
- **Platform:** Shopify

## Features

- Responsive design optimized for all devices
- Modern gaming-focused aesthetics
- Advanced product comparison functionality
- Wishlist capabilities
- Newsletter integration
- Instagram gallery integration
- Team member showcase
- Testimonials with video support
- Advanced search and filtering
- Multi-language support
- SEO optimized

## Installation

1. Download the theme files
2. Upload to your Shopify store via the admin panel
3. Customize the theme settings through the Shopify theme editor

## Customization

The theme includes extensive customization options through the Shopify theme editor:

- Color schemes
- Typography settings
- Layout options
- Section configurations
- Footer customization
